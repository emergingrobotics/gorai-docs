## About This Book

This book teaches you to build robot software using Gorai, a Go-based robotics framework that uses NATS for messaging. You'll learn:

- Gorai's architecture and design philosophy
- NATS messaging patterns for robotics
- Building sensors, actuators, and services
- Testing strategies for robot software
- AI/ML integration for edge inference
- Project organization and best practices

**Prerequisites**: Basic Go programming knowledge. Familiarity with Linux command line helpful but not required.

**Code examples**: All code examples are available in the Gorai repository.
