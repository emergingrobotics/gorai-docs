# Hello Sensor Example

A complete walkthrough of building a temperature sensor component.

## Overview

This example builds a CPU temperature sensor that:

- Reads the CPU temperature from the Linux thermal zone
- Publishes readings to a NATS topic
- Supports configuration and reconfiguration
