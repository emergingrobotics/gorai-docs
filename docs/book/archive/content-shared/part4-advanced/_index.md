# Part IV: Advanced Topics

> **In This Part:** Integrate AI/ML models, organize large projects, leverage AI-assisted development, and chart your path forward.

## Overview

Part IV covers advanced topics that will take your GoRAI projects to the next level. You'll learn to run ML models on edge hardware, organize code for growing projects and teams, use AI tools to accelerate development, and understand where GoRAI is headed.

These chapters are more independent than earlier parts—read them based on your interests and needs.

## Chapters in This Part

| Chapter | Title | What You'll Learn |
|---------|-------|-------------------|
| [Chapter 14](ch14-ai-ml/_index.md) | AI/ML Integration | NPU/GPU acceleration, model deployment, vision ML |
| [Chapter 15](ch15-organization/_index.md) | Project Organization | Monorepos, separate repos, fleet management |
| [Chapter 16](ch16-ai-dev/_index.md) | AI-Assisted Development | Using Claude, Copilot, and other AI tools effectively |
| [Chapter 17](ch17-conclusion/_index.md) | Conclusion | Summary, roadmap, community, next steps |

## Key Concepts Introduced

By the end of Part IV, you'll understand:

- **Hardware acceleration**: NPU (RK3588), GPU (NVIDIA), TPU (Coral) integration
- **Model deployment**: ONNX, quantization, inference pipelines
- **Code organization**: When to use monorepos vs separate packages
- **AI-assisted workflows**: Effective prompting for robotics code
- **Community involvement**: How to contribute to GoRAI

## Prerequisites

Part IV assumes you've worked through Parts I-III. Specific chapters have additional requirements:

- **Chapter 14 (AI/ML)**: Familiarity with ML concepts helpful but not required
- **Chapter 15 (Organization)**: Experience with Go modules and git
- **Chapter 16 (AI-Assisted Dev)**: Access to an AI coding assistant (Claude, Copilot, etc.)

## Reading Order

These chapters are largely independent. Read based on your interests:

- **Running ML on robots?** Start with Chapter 14
- **Growing your codebase?** Start with Chapter 15
- **Using AI tools?** Start with Chapter 16
- **Planning next steps?** Read Chapter 17
