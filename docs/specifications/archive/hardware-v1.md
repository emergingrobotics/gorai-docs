# Hardware Specifications

This document outlines compatible hardware for the Gorai robotics platform.

**Note:** None of this hardware is required — these are simply components known to work at one point in time. Amazon links are provided for convenience only; many other providers often have better prices.

## Single Board Computers

Full-featured SBCs suitable for running the main Gorai stack with Linux.

| Board | Notes |
|-------|-------|
| Raspberry Pi 3 | Widely available, good community support |
| Raspberry Pi 4 | Improved performance, USB 3.0, Gigabit Ethernet |
| Raspberry Pi 5 | Latest generation, significant performance improvements |
| Raspberry Pi Zero | Compact form factor, lower power consumption |
| BeagleBone family | Industrial-grade options, real-time capabilities |
| [Orange Pi 5B Plus](https://www.amazon.com/dp/B0DDTJS7BT) | Rockchip RK3588S, high performance |
| [Radxa Rock 3A](https://www.amazon.com/components-Single-Computer-Rockchip-RK3568/dp/B0BV324L5P/) | Rockchip RK3568, good balance of price/performance |

## Tiny Single Board Computers

Compact SBCs with NPU capabilities, ideal for edge AI inference in robotics applications.

| Board | Notes |
|-------|-------|
| [RV1103-based Tiny Board](https://www.amazon.com/dp/B0CMCWC9DG) | Ultra-compact, integrated NPU |
| [RV1106 with PoE](https://www.amazon.com/dp/B0D6G7Z2BQ) | Power over Ethernet support |
| [RV1103 with Ethernet](https://www.amazon.com/dp/B0D6NJFNGX) | Wired networking capability |
| [Lyra Plus 3506 with Ethernet](https://www.amazon.com/dp/B0DRFS62MY) | Enhanced connectivity options |
