    Installation#

    This guide covers installing Gorai and its dependencies on your development machine.

    Prerequisites#

    • Go 1.21+ — Download Go
    • NATS Server — Local or remote NATS installation
    • Git — For cloning repositories

    Install Go#

    macOS#

    brew install go

    Ubuntu/Debian#

    sudo apt update
sudo apt install golang-go

    Verify Installation#

    go version
# go version go1.21.0 linux/amd64

    Install NATS#

    Option 1: Binary#

    # macOS
brew install nats-server

# Linux
curl -L https://github.com/nats-io/nats-server/releases/download/v2.10.0/nats-server-v2.10.0-linux-amd64.tar.gz | tar xz
sudo mv nats-server-v2.10.0-linux-amd64/nats-server /usr/local/bin/

    Option 2: Docker#

    docker run -p 4222:4222 nats:latest

    Verify Installation#

    nats-server --version
# nats-server: v2.10.0

    Install Gorai CLI#

    go install github.com/gorai/gorai/cmd/gorai@latest

    Verify:

    gorai version

    Next Steps#

    With everything installed, proceed to the Quick Start to build your first component.
