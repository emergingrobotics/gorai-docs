## Project Comparison

| Project | Skills Learned | Hardware Cost | Time |
|---------|----------------|---------------|------|
| [Hello Sensor](hello-sensor.md) | Core patterns, pub/sub, fake sensors | $0 | 30 min |
| [Pan-Tilt](pan-tilt.md) | Real hardware, servos, cameras, sensor fusion | $150-350 | 2-4 hours |
| [Skimmer](skimmer.md) | Autonomous navigation, GPS, motor control | ~$530 | 1-2 days |

## Suggested Order

1. **Start with Hello Sensor** - No hardware needed, teaches core concepts
2. **Build Pan-Tilt** - Add real hardware integration
3. **Create Skimmer** - Full autonomous system
