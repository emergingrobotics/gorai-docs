# Preface {.unnumbered}

Welcome to *Gorai: Building Modern Robots with Go and NATS*.

This book is a practical guide to building robotics software using Go, NATS messaging, and modern AI integration. Whether you're a software developer curious about robotics, a robotics enthusiast looking for a modern approach, or an experienced engineer seeking something better than existing frameworks, this book will help you get started.

## Why We Wrote This Book

We believe robotics software should be simpler. The tools exist—Go's elegant concurrency, NATS's battle-tested messaging, modern AI accelerators—but nobody had put them together in a way that prioritized developer experience. Gorai is our answer, and this book is your guide to using it.

## How to Read This Book

**New to robotics?** Read sequentially from the beginning. Each chapter builds on the previous.

**Experienced with ROS?** Skim Part I for philosophical differences, then dive into Chapter 4 (NATS) and Chapter 12 (Hello Sensor).

**Go developer?** Part I (architecture) and Part II (components) will orient you quickly.

**Just want to build?** Start with Chapter 11 (Development Environment) and Chapter 12 (Hello Sensor). Get code running, then circle back.
